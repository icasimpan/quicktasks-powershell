Install
```
brew cask install powershell
```

Run
```
pwsh
```

To update/upgrade later on:
```
brew update
brew cask upgrade powershell
```

See https://docs.microsoft.com/en-us/powershell/scripting/install/installing-powershell-core-on-macos?view=powershell-7
